Enfoque de la Arquitectura

La arquitectura seleccionada para el sistema del Hotel Cacique Bolívar se basa en el patrón Modelo-Vista-Controlador (MVC), implementado con Spring Boot y Java. Este enfoque permite separar de forma clara la lógica de negocio, la interfaz de usuario y el control de flujo, lo que facilita el mantenimiento y la escalabilidad del proyecto.

Elección de Estilo

El estilo arquitectónico elegido es Arquitectura en Capas bajo el patrón MVC. En este esquema:

  • Modelo: gestiona los datos y reglas de negocio (reservas, clientes, paquetes, pagos).
  • Vista: ofrece una interfaz web accesible desde dispositivos móviles y de escritorio.
  • Controlador: actúa como intermediario entre la vista y el modelo, procesando solicitudes y devolviendo respuestas.

Justificación

La elección del patrón MVC con Spring Boot responde a varios factores clave:

  • Mantenibilidad: la separación de responsabilidades reduce la complejidad y permite modificar módulos de forma independiente.
  • Escalabilidad: el sistema puede crecer incorporando nuevas funcionalidades (paquetes turísticos, PQRS+) sin afectar los módulos existentes.
  • Compatibilidad: al ser una aplicación web, garantiza acceso desde cualquier dispositivo con conexión a Internet.
  • Productividad: Spring Boot proporciona herramientas y configuraciones que agilizan el desarrollo y la integración con bases de datos.
